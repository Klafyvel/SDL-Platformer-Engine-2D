# SDL-Platformer-Engine-2D
All I need to create a platformer with SDL simply.

## Features ##

- [ ] 2D Physic engine
    - [ ] Bases
    - [ ] Collision
    - [ ] Gravity
- [ ] Graphic module
    - [ ] Basic window + sprites
    - [ ] Animable objects
- [ ] Control module
    - [ ] Keys
    - [ ] easy mainloop
- [ ] Audio module
- [ ] Menu module (to be adapted from [here](https://github.com/Klafyvel/Menu_API))
